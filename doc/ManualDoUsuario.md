# Manual do Usuário – Sistema de Controle de Estoque (SCE)

Bem-vindo ao Sistema de Controle de Estoque (SCE).  
Este manual tem como objetivo orientar o usuário sobre o funcionamento básico do sistema.

---

## Passo a Passo de Uso

1. Abrir o sistema. Ao iniciar, um menu com as opções principais será exibido.  
2. Cadastrar Produto. Informe nome, quantidade e valor unitário para adicionar um novo item ao estoque.  
3. Registrar Entrada. Escolha um produto existente e informe a quantidade a ser adicionada.  
4. Registrar Saída. Escolha um produto e informe a quantidade a ser retirada do estoque.  
5. Relatório de Estoque. Visualize todos os produtos cadastrados com suas quantidades e valores atuais.  
6. Encerrar o sistema. Digite “0” para sair.

---

## Requisitos de Operação
- O sistema é fictício e executado em ambiente simulado.  
- Todos os dados são armazenados apenas enquanto o programa estiver em execução.
